# Contributor Code of Conduct

We value the participation of every contributor and want to maintain an open and welcoming project.

This project follows the Contributor Covenant v2.1. Please read the full text at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

If you experience or witness unacceptable behavior, please report it to the project maintainers at replace-with-your-email@example.com. All reports will be handled with discretion.

Key points
- Be respectful and considerate in all communications.
- Be collaborative and assume good intent.
- Harassment and discrimination will not be tolerated.

For any questions about this code of conduct, contact the maintainers.
